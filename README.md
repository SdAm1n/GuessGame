# GuessGame
This is a simple number Guessing Game written in C++.

## How to run on Linux
* Using the makefile
1. open terminal in project directory
2. use the following command:
3. $ make
4. $ ./'Guess Game'

* Without using makefile : can use the following commands to compile the project :
1. $ g++ *.cpp -o filename
2. $ ./filename
